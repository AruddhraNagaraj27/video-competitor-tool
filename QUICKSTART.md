# QUICK START - 3 MINUTE SETUP

## For the Impatient (TL;DR)

```bash
# 1. Get YouTube API Key
# Go to: https://console.cloud.google.com/
# - Create project
# - Search "YouTube Data API v3"
# - Enable API
# - Create API Key
# - Copy key

# 2. Clone/Download Code
# Download this folder or clone from GitHub

# 3. Backend Setup
cd server
npm install
cp .env.example .env
# Edit .env and paste your YouTube API key
npm start
# Server running on http://localhost:5000

# 4. Frontend Setup (new terminal)
cd client
npm install
npm run dev
# Open http://localhost:3000

# 5. Test
# Enter: Amul
# Competitors: Aavin, Country Delight, Milky Mist, Akshayakalpa
# Click Generate Report
# Download PowerPoint
```

## That's It!

Your app is now running locally.

To deploy to production:

1. Push to GitHub
2. Deploy backend to Render (takes 2 minutes)
3. Deploy frontend to Vercel (takes 2 minutes)
4. Your live URL is ready!

See `SETUP_GUIDE.md` for detailed deployment steps.
