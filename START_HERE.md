```bash

npm run dev

# Open http://localhost:3000
```

### Option 2: Deploy to Production (15 minutes)

1. **Get YouTube API Key** (3 min)
   - Visit: https://console.cloud.google.com/
   - Create project → Enable YouTube Data API v3 → Create API Key

2. **Deploy Backend to Render** (5 min)
   - Push code to GitHub
   - Create Render account
   - Connect GitHub → Deploy with API key
   - Copy Render URL

3. **Deploy Frontend to Vercel** (5 min)
   - Create Vercel account
   - Import GitHub project
   - Add Render URL as environment variable
   - Deploy

**Result**: Live app at `https://your-app.vercel.app` 🎉

## 📊 HOW IT WORKS

### Step 1: Input

User enters company name + 4 competitors

### Step 2: Fetch Data

Backend uses YouTube API to get:

- Channel information
- 50 latest videos
- View/like/comment counts
- Upload dates
- Video durations

### Step 3: Analyze

Backend analyzes:

- Engagement rates
- Posting frequency
- Content themes
- Performance metrics
- Competitive positioning

### Step 4: Score

Each company gets scored 0-100:

- Subscriber Strength (20 pts)
- Views Performance (20 pts)
- Engagement Rate (20 pts)
- Posting Consistency (20 pts)
- Content Variety (20 pts)

### Step 5: Generate Report

PowerPoint with 11 slides:

1. Cover slide
2. Executive summary
3. Channel overview
4. Top videos
5. Engagement analysis
6. Posting frequency
7. Content themes
8. Gap analysis
9. Recommendations
10. Final ranking
11. Key takeaways

### Step 6: Display & Download

- Interactive web report (9 sections)
- One-click PowerPoint download
- Professional formatting
