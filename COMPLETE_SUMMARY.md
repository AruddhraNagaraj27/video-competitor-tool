## 🎯 WHAT THE APP DOES

### User Experience

```
1. User opens app
2. Enters company name + 4 competitors
3. Clicks "Generate Report"
4. System fetches YouTube data
5. Shows interactive web report
6. User can download PowerPoint
7. Gets professional 11-slide presentation
```

### Report Contents

- Executive summary
- Channel comparison
- Top performing videos
- Engagement analysis

## 📋 QUICK START GUIDE

### For Impatient People (5 minutes)

```bash
# 1. Get API key
# Visit: https://console.cloud.google.com/
# Search "YouTube Data API v3" → Enable → Create Key

# 2. Clone/Download code
# This folder: video-competitor-tool/

# 3. Backend
cd server
npm install
echo "YOUTUBE_API_KEY=your_key_here" > .env
npm start

# 4. Frontend (new terminal)
cd client
npm install
npm run dev

# 5. Open http://localhost:3000
# Done! Test with demo companies above.
```
